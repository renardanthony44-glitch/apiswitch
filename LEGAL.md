# Conditions Générales d'Utilisation – ApiSwitch

## 1. Responsabilité Partagée
ApiSwitch agit comme un routeur passif : nous ne modifions pas, ne stockons pas, ni n'analysons le contenu des requêtes/réponses API transitant par notre service.
**Votre responsabilité** : Vérifier que vos clés API sont valides et que leur usage respecte les CGU des fournisseurs tiers (ex: OpenAI, Stripe).
**Notre responsabilité** : Maintenir la disponibilité du service (SLA : 99% pour les tiers payants).

## 2. Neutralité de l'Outil
ApiSwitch est agnostique : nous ne favorisons aucun fournisseur d'API. Le basculage entre clés est basé sur la disponibilité, le coût, ou la latence.

## 3. RGPD & Protection des Données
- Aucune donnée personnelle n'est collectée par défaut.
- Self‑hosted : Vous pouvez héberger ApiSwitch sur vos serveurs (code open‑source sous licence MIT).
- Logs : Les métriques sont anonymisées. Les adresses IP sont **truncées / anonymisées après 24 h** (sauf en mode auto‑hébergement où vous contrôlez le stockage).
